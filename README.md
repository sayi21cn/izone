# 必须环境

- Mysql
- JDK1.6+
- Maven

# 安装说明

新建数据库 `izone`，字符集为 `UTF-8`，导入 `izone.sql` 到你的数据库。

```sh
$ git clone https://github.com/junicorn/izone.git
$ cd izone
$ jetty:run
```
